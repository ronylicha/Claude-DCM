The Problem

Claude Code forgets.
DCM remembers.

When context compaction hits, your session state vanishes. Tasks disappear. Agents lose coordination. You start over.

⚠ Without DCM

  • Active tasks vanish after compaction
  • Agents work in silos with no visibility
  • Dangerous commands execute unchecked
  • Manual re-explanation every session
  • No audit trail of decisions or actions

🛡 With DCM

  • Full state restored automatically after compaction
  • Inter-agent pub/sub messaging & coordination
  • Safety gate blocks destructive operations
  • Context briefs injected seamlessly on resume
  • Every action logged with complete lineage
Features

Everything your agents need.

A complete orchestration layer built from the ground up for multi-agent Claude Code workflows.

🛡

Safety Gate

PreToolUse hook that blocks destructive operations before execution. Filesystem wipes, database drops, secret exposure — all caught and logged.

Critical
🧠

Context Guardian

4-layer defense system: local monitor, API health checks, stop guard, and compact save/restore. Your session survives compaction intact.

Core
📡

Inter-Agent Messaging

Built-in pub/sub with topics, subscriptions, and blocking dependencies. Agents communicate, coordinate, and share results automatically.

Orchestration
🌊

Wave Orchestration

Decompose complex work into sequential phases with parallel subtasks. State machine tracks progress, detects conflicts, synthesizes results.

Orchestration
📊

Real-Time Dashboard

11-page monitoring UI with live activity feed, SVG topology visualization, wave timeline, KPI chips, and performance metrics.

Monitoring
🧭

Intelligent Routing

Learns which tools work best for each task through feedback-driven keyword scoring. Routes evolve dynamically based on success rates.

Intelligence
🔄

Compact Restore

Full snapshot saved before compaction — tasks, agents, files, decisions, wave state. Restored as structured Markdown brief on resume.

Recovery
🔍

Complete Audit Trail

Every tool invocation, message, decision, and state change logged with full parent/child hierarchy. PostgreSQL-backed with JSONB metadata.

Observability
⚡

WebSocket Events

Real-time event streaming with HMAC authentication and channel subscriptions. Dashboard updates instantly via PostgreSQL LISTEN/NOTIFY bridge.

Quick Start

Up and running in 60 seconds.

One CLI handles everything: dependencies, database, hooks, and services.

terminal 
# Clone the repository
git clone https://github.com/ronylicha/Claude-DCM.git
cd Claude-DCM/context-manager

# Install everything (deps, DB, hooks, supervisor)
./dcm install

# Start all 3 services
./dcm start

# Verify health
./dcm status

# Open the dashboard
open http://localhost:3848

Stop losing context.
Start building faster.
